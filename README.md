# 🌆 Jönköping City Guide

A modern iOS City Guide application designed to help users discover, organize, and manage local experiences in Jönköping. The app acts as a centralized digital companion for tourists and residents, combining attractions, restaurants, hotels, and personal planning tools in one seamless interface.

---

## 📱 About the App

**Jönköping City Guide** is a prototype iOS application built to solve the problem of fragmented city information. Instead of searching across multiple websites and platforms, users can explore everything in one place.

The app uses **mock hosted data** to simulate a real-world city guide experience.

---

## 🎯 Target Users

This app is designed for:
- 🧳 Tourists visiting Jönköping
- 🏙️ Local residents exploring city life
- 📅 Users who want to organize and plan activities easily

---

## ❗ Problem It Solves

Many cities suffer from scattered information across multiple platforms.

This app solves that by:

- 📦 **Centralizing Data**  
  Hotels, restaurants, attractions, and stores are unified in one interface using a mock API.

- ⏰ **Preventing Forgetfulness**  
  Local notifications remind users about upcoming plans.

- 🗂️ **Personal Organization**  
  Users can create custom collections (folders) like *“Summer 2026”* to save and organize events.

---

## ✨ Key Features

### 🃏 Interactive Discovery Cards
- Flippable UI cards
- Front side: image preview
- Back side: detailed location & info

### 🔔 Smart Notification System
- One-tap scheduling
- Automatic reminders 24 hours before events

### 📚 SwiftData Collections
